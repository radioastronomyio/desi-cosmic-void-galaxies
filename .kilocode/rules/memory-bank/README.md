### Session Start
1. Load `brief.md` for foundational context
2. Load `context.md` for current state and next steps
3. Reference other files as needed

### Session End
1. Update `context.md` with accomplishments and next steps
2. Update other files if relevant changes occurred
3. Commit memory-bank updates with main work
